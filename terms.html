Сайт (Web-сайт) — совокупность программно-аппаратных средств для ЭВМ, обеспечивающих публикацию для всеобщего обозрения данных о Заказчике в интернете. Сайт доступен по уникальному электронному адресу или его буквенному обозначению. Может содержать графическую, текстовую, аудио-, видео-, а также иную информацию, записанную и читаемую с помощью ЭВМ. В настоящем Договоре под Сайтом понимается сайт Заказчика. Сайт — это система, включающая «дизайн» и «программирование», позволяющая публиковать информацию в сети Интернет и предоставлять к ней доступ по определённым сценариям. Требования к Сайту содержатся в Концепции и ТЗ. Стоимость услуг по разработке сайта определяется в Приложении 1. В понятие сайта не входит информационное наполнение (контент).

Доменное имя — символьное обозначение, предназначенное для адресации сайта в сети Интернет и удобства обеспечения доступа к информации, размещённой на сайте.

Шаблон сайта — это структура подготовленных файлов (макет, HTML и CSS-код), а также дизайн-схемы сайта, определяющая визуальное представление страниц сайта. Шаблон может быть создан с нуля или взят за основу из готового решения.

Готовое решение Битрикс — это программный продукт, в который включены типовые шаблоны и модули, устанавливаемые на сайт на CMS 1С-Битрикс.

CMS (движок сайта, движок) — программное обеспечение для управления сайтом и его контентом. Обеспечивает создание, редактирование, удаление страниц, управление пользователями и настройками сайта. Примеры CMS: WordPress, 1С-Битрикс, Opencart и др.

1С-Битрикс — одна из самых популярных российских систем управления содержимым сайтов и интернет-магазинов. Один из лидеров рынка коммерческих CMS в России.

Opencart — это бесплатная и открытая платформа для создания интернет-магазинов и сайтов-каталогов товаров и услуг. Отличается простотой установки и управления, большим количеством готовых модулей и шаблонов.

WordPress — система управления содержимым сайтом (CMS) с открытым исходным кодом. Бесплатно распространяется и используется для создания сайтов, блогов, интернет-магазинов и других веб-проектов. Для WordPress существует множество бесплатных и платных тем и плагинов.

Opensours — программное обеспечение с открытым исходным кодом, позволяющее использовать, изменять и распространять код свободно (open source).

ПО — программное обеспечение, совокупность программ для работы компьютера, сайта, сервера и т.д.

Woocommerce — модуль для WordPress, позволяющий создавать интернет-магазины и управлять продажами товаров, совместим с WordPress.

Хостинг сайта — услуга по предоставлению серверного дискового пространства и ресурсов, чтобы сайт имел постоянный доступ онлайн (обычно Интернет).

Доменное имя - обозначение символами, предназначенное для адресации сайтов в сети "Интернет" в целях обеспечения доступа к информации, размещенной в сети "Интернет";

Сайт использует данные cookie для сервиса веб-аналитики Яндекс.Метрика. Продолжая использовать данный сайт, вы соглашаетесь с Политикой в отношении обработки персональных данных.
